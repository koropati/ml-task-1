Introduction

Background

  • Synthetic Aperture Radar (SAR) imaging provides high-resolution images regardless of weather or lighting conditions
  • SAR image target classification is crucial for military and civilian applications
  • Applications include marine surveillance, fisheries management, and emergency response

Challenges

  • Traditional methods have limited accuracy
  • Deep learning methods are computationally intensive and inefficient
  • Need for both high accuracy and efficiency in SAR image classification

"Fast and accurate target classification is important in both military and civilian fields."

Why SAR Imaging?

SAR can operate in all weather conditions, day or night, providing consistent imaging capabilities unlike optical systems.

Research Problem & Objectives

Problem Statement

  • Traditional machine learning methods for SAR image classification have limited accuracy
  • Deep learning methods achieve high accuracy but suffer from low efficiency
  • There is a need for methods that balance both accuracy and efficiency

Research Objectives

  • Develop a SAR image classification method that achieves both high accuracy and efficiency
  • Extract effective features using Gabor wavelets
  • Apply dimensionality reduction with PCA
  • Implement an efficient K-NN classifier with appropriate distance metrics

Overview of the Proposed Method

Apply Principal Component Analysis to reduce feature dimensions and avoid the dimensionality disaster.

3

K-NN Classification

Train a K-Nearest Neighbor classifier using cosine similarity as the distance metric.

Gabor Feature Extraction

Why Gabor Wavelets?

  • Excellent for extracting local spatial and frequency domain information
  • Fewer data to process compared to other methods, enabling real-time processing
  • Can handle image rotation and deformation, improving system robustness
  • Sensitive to edge features in SAR images
  • Captures spatial localization, orientation selectivity, and spatial frequency selectivity

Implementation Details

  • Gabor kernels set at seven scales and ten orientations
  • Image size: 128 × 128 pixels
  • Images divided into sub-graphs of 31 × 31 pixels
  • Energy of each sub-graph calculated
  • Energy matrices combined into a row vector
  • Final feature dimension: 7700

Figure 2: 10 orientations of Gabor features

Gabor Feature Formula

The Gabor feature extraction is the convolution of the image with the Gabor kernels.

Principal Component Analysis (PCA)

Why PCA?

Dimensionality Disaster

When the number of features exceeds a certain threshold, classifier performance degrades rather than improves - known as the "dimensionality disaster" in pattern recognition.

PCA Solution

PCA maps high-dimensional data to low-dimensional space while preserving as much information as possible.

Implementation Details

  • Original Gabor feature dimension: 7700
  • Applied PCA with 80% compression ratio
  • Reduced feature dimension to 1540
  • Features with higher distinction retained with larger weights
  • Features lacking distinction given lower weights

K-NN Classification

Why K-NN?

  • K-NN depends on neighboring finite samples rather than class fields
  • More suitable for datasets with overlapping or cross-class fields (like MSTAR)
  • Simple implementation yet effective classification
  • Works well with properly selected features

Distance Metric: Cosine Similarity

similarity = cos(θ) = (A × B) / (||A|| ||B||)

  • Cosine similarity focuses on orientation differences rather than absolute values
  • More suitable for SAR image data diversity than Euclidean or Minkowski distance
  • Measures the cosine of the angle between two vectors (feature sets)
  • Values range from -1 (opposite) to 1 (same direction)

K-NN Classification Process

1

Calculate cosine similarity between test sample and all training samples

2

Identify k nearest neighbors based on similarity values

3

Classify test sample based on majority class among neighbors

Implementation Detail

The number of neighbors (k) was set to 10 through cross-validation to achieve optimal performance.

Results Analysis

  • The proposed method achieved the highest overall accuracy at 99.41%
  • Next best method was SFA at 95.75% (3.66% lower)
  • Perfect classification (100%) for T72 class
  • BMP2 and BTR70 achieved 99.15% and 98.47% accuracy, respectively
  • Significant improvement over all compared methods

Efficiency Analysis

Computational Efficiency

  • Gabor feature extraction is computationally lightweight
  • PCA reduces dimensionality by 80%, further improving efficiency
  • K-NN classification is simple and fast compared to deep learning methods

Training Efficiency

  • No complex training process like in deep learning
  • Feature engineering approach requires less training data
  • Parameter tuning is straightforward and efficient

Key Success Factors

1. Effective Feature Extraction

Gabor wavelets are particularly suited for capturing the edge and texture characteristics of SAR images. The multi-scale and multi-orientation approach ensures comprehensive feature representation.

2. Optimal Feature Selection

PCA with 80% compression ratio strikes the perfect balance between dimensionality reduction and information preservation, avoiding the dimensionality disaster.

3. Appropriate Classifier Selection

K-NN with cosine similarity is well-suited for SAR image classification, particularly for datasets with overlapping class fields like MSTAR.

Conclusion

Summary of Contributions

  • Proposed a novel SAR image classification method combining Gabor feature extraction, PCA, and K-NN
  • Achieved superior performance in terms of both accuracy (99.41%) and efficiency
  • Successfully balanced the trade-off between accuracy and computational efficiency
  • Demonstrated that feature engineering approach can outperform deep learning methods for SAR image classification

Key Findings

Gabor Features

Gabor features at seven scales and ten orientations provide optimal representation of SAR image characteristics.

PCA Compression

80% compression ratio provides the best balance between dimension reduction and information preservation.

Distance Metric

Cosine similarity significantly outperforms Euclidean and Minkowski distances for SAR image classification.

Limitations and Future Work

Limitations

  • Tested only on three classes of targets
  • Performance under different noise conditions not explored
  • Limited to SAR imagery, may not generalize to other imaging modalities

Future Research Directions

  • Extend the method to more target classes and complex scenarios
  • Investigate robustness against noise and environmental conditions
  • Integrate with other feature types for enhanced performance
  • Explore hybrid approaches combining feature engineering with deep learning
  • Apply to real-time SAR image classification systems
